# Metronomos

A collection of crates to handle dependency injection and lifecycle management in Rust applications.

The main crate is `metronomos`, which provides a simple and flexible way to manage dependencies and their lifecycles.
It allows you to define services, register them, and resolve them when needed.

Other crates include:
- `metronomos-pulse` library crate for managing the dependency initialization and resolution process.
- `metronomos-loom` library crate for generating the dependency graph to resolve dependencies in the correct order.

The project is open-source and welcomes contributions from the community.
